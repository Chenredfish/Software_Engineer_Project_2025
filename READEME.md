# 網站架設及如何使用

## 第一步:安裝node.js
- 檢查是否有安裝
    node -v
    npm -v
    有版本號就可以進行下一步
## 第二步:啟動後端
- cd Software_Engineer_Project_2025/backend  //依據你Backend資料夾所在地進行調整
- npm install
- npm run dev

## 第三步:啟動前端
- cd Software_Engineer_Project_2025/frontend
- npm install
- npm install react-router-dom   //切換頁面用
- npm install axios   //呼叫後端API
- npm install @mui/material @emotion/react @emotion/styled     //MUI
- npm start
